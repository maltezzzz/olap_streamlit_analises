# Streamlit OLAP - DW Combustiveis v4

Estrutura do projeto gerado para Victor — app com páginas separadas:
- Página "Pivot OLAP"
- Página "Dashboard"
- Página "Séries Temporais"
- Página "Exportar / Detalhes (Drill-through)"

Instruções:
1. Ajuste as credenciais em `db.py`.
2. Instale dependências: `pip install -r requirements.txt`
3. Rode: `streamlit run app.py`
